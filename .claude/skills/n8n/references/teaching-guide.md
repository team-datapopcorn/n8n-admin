# 교육 가이드

n8n 고급 활용법을 교육하기 위한 콘텐츠 생성 모듈. 대상: n8n 기본을 아는 사용자.

## 출력 형태

사용자가 원하는 출력 형태를 확인한다. 지정하지 않으면 마크다운 튜토리얼을 기본으로 한다.

### 1. 마크다운 튜토리얼

구조:
```
# [주제명]

## 개요
- 이 튜토리얼에서 배울 내용
- 사전 요구사항

## 핵심 개념
- 관련 개념 설명 (간결하게)

## 단계별 구현

### Step 1: [노드/설정 이름]
- 왜 이 노드를 사용하는지
- 설정 방법
- 주의사항

### Step 2: ...

## 완성된 워크플로우 요약
- 전체 노드 구성 다이어그램 (텍스트)
- 데이터 흐름 설명

## 응용 패턴
- 이 패턴을 변형할 수 있는 사례
- 추가 학습 추천
```

저장 위치: `docs/tutorials/n8n/[주제명].md`

### 2. 실습형 (워크플로우 직접 생성)

- 설명과 동시에 실제 워크플로우를 n8n에 생성한다
- 각 단계마다 워크플로우를 점진적으로 구성:
  1. 기본 트리거 노드만 있는 워크플로우 생성 (`n8n_create_workflow`)
  2. 설명 후 노드 추가 (`n8n_update_partial_workflow`)
  3. "지금 n8n UI에서 워크플로우를 열어 확인해보세요" 안내
  4. 다음 노드 추가 및 연결
  5. 완성 후 테스트 (`n8n_test_workflow`, 가능한 경우)

- 각 단계 사이에 사용자가 이해했는지 확인:
  - "여기까지 이해되셨나요? 다음 단계로 넘어갈까요?"

### 3. 강의 슬라이드용

구조화된 콘텐츠를 슬라이드 형식으로 출력한다.

```
## 슬라이드 1: [제목]
**본문:**
- 핵심 포인트 1
- 핵심 포인트 2

**발표자 노트:**
- 부연 설명
- 예시

---

## 슬라이드 2: [제목]
...
```

- `ppt-design-system` 스킬이 사용 가능하면 연계 안내 (선택적)
- 다이어그램이 필요하면 `svg-diagram` 스킬 연계 안내 (선택적)
- 두 스킬이 없어도 텍스트 기반으로 완전한 콘텐츠 출력

## 교육 주제 카테고리

### 고급 노드 활용
- Code 노드: JavaScript/Python으로 커스텀 로직 작성
- Function 노드: 데이터 변환 및 가공
- HTTP Request 심화: 인증, 페이지네이션, 에러 처리
- Webhook 노드: 외부 서비스 연동 엔드포인트

### 워크플로우 설계 패턴
- 에러 핸들링: Error Trigger + 알림 체인
- 재시도 패턴: 실패 시 자동 재실행 로직
- 서브 워크플로우: 공통 로직 분리 및 재사용
- 조건 분기 고급: Switch + Merge 패턴

### 외부 서비스 연동
- API 인증 방식: API Key, OAuth2, Bearer Token
- 웹훅 수신: 외부 서비스의 이벤트를 n8n으로 수신
- 커스텀 API 호출: REST API 설계 패턴

### 성능 최적화
- SplitInBatches: 대량 데이터 배치 처리
- 메모리 관리: 대용량 데이터 처리 시 주의사항
- 실행 시간 최적화: 불필요한 노드 제거, 병렬 처리

## 콘텐츠 생성 원칙

1. **실용 우선**: 이론보다 바로 적용 가능한 예제 중심
2. **점진적 복잡도**: 간단한 예제에서 시작해 점점 복잡하게
3. **왜(Why) 설명**: 단순 방법이 아니라 왜 이렇게 하는지 설명
4. **주의사항 명시**: 자주 발생하는 실수와 해결법 포함
